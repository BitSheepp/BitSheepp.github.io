# 你只需要做这几步（中文）

## 1) 本地先看效果
在 `personal-cv-site` 目录运行：

```bash
python -m http.server 8000
```

浏览器打开 `http://localhost:8000`。

## 2) 上传到 GitHub
由于当前机器没有 `gh` 命令，我给你最稳的 git 方式：

```bash
cd personal-cv-site
git init
git add .
git commit -m "init: cv website"
git branch -M main
git remote add origin https://github.com/<你的用户名>/<你的仓库名>.git
git push -u origin main
```

## 3) 打开 GitHub Pages
GitHub 仓库 -> Settings -> Pages -> Source 选择 `GitHub Actions`。

之后每次你改 `data/profile.json`，再 `git add/commit/push`，网站会自动更新。

## 4) 手动更新内容（最关键）
只改一个文件：
- `data/profile.json`

不需要改 `index.html`。
