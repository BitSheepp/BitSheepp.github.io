# 论文更新说明（CSV驱动）

你后续每次更新论文，只需要：

1. 从 Google Scholar 导出 CSV，覆盖到固定路径：
   `D:\0_实验室\0-hcy-cv\文献引用\citations.csv`

2. 在项目目录执行：

```powershell
cd C:\Users\1\Documents\Playground\personal-cv-site
python .\tools\generate_publications_from_csv.py
```

3. 推送网站：

```powershell
git add .
git commit -m "update publications from scholar csv"
git push
```

排序规则：
- 已发表在前，未发表在后
- 同组按影响因子从高到低
- 再按年份从新到旧

影响因子映射位置：
- `tools/generate_publications_from_csv.py` 中的 `JOURNAL_IF`

你提供 DOI / PDF 后，我会帮你写入 `data/publications.json` 对应条目，页面会自动显示下载按钮。
